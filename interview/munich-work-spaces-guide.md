## 1. FREE: Public Libraries (Münchner Stadtbibliothek)

All Munich public libraries offer: **Free WiFi (Hotsplots)**, workspaces, and heating. No membership needed for WiFi access.

### Bayerische Staatsbibliothek (State Library) - TOP RECOMMENDATION

**Address:** Ludwigstraße 16, 80539 München (U3/U6 Universität)

| Feature | Details |
|---------|---------|
| **Total Workspaces** | ~850 seats |
| **Best Area** | "Plaza" - 130 modern workspaces, carrels (private booths), lounge |
| **WiFi** | Free BayernWLAN (no registration needed) |
| **Hours** | Allgemeiner Lesesaal: Mon-Fri until 20:00, Sat 9:00-17:00 |
| **Registration** | Free library card available at ground floor |

**Why it's great for interviews:**
- Private carrels (small booths) for video calls
- Modern "Plaza" area with diverse seating options
- Professional atmosphere
- Very quiet

**Note:** As of March 2025, no reservation required for reading rooms.

[Website](https://www.bsb-muenchen.de/recherche-und-service/besuche-vor-ort/lesesaelearbeitsplaetze/)

## Interview Call Checklist

### Before the Call:
- [ ] Test WiFi speed (need 5+ Mbps for video)
- [ ] Scout the location the day before
- [ ] Check background - plain wall is best
- [ ] Test audio with earbuds/headphones
- [ ] Have backup hotspot ready (phone)

### Location Requirements for Video Interviews:
- [ ] **Quiet** - no background chatter
- [ ] **Good lighting** - face a window if possible
- [ ] **Stable WiFi** - test with a speed test
- [ ] **Power outlet** - don't risk battery dying
- [ ] **Professional background** - or use virtual background
- [ ] **Backup plan** - what if WiFi fails?

### Best Time to Use Libraries:
- **Quietest:** Weekday mornings (10:00-12:00)
- **Busiest:** Afternoons and weekends (avoid for calls)

## Recommended Strategy for Your Pliant Interview

### Option A: Library with Private Carrel (Free)
1. Go to **Bayerische Staatsbibliothek**
2. Get a free library card at ground floor (takes 10 min)
3. Reserve a **Carrel in the Plaza** - these are semi-private booths
4. Test the WiFi the day before
5. Arrive 30 min early to settle in

### Option B: Coworking Phone Booth (Free Trial)
1. Book a **free trial day at WERK1** in advance
2. Arrive early, get oriented
3. Use the **phone booth** for your 30-min call
4. Professional environment, good backup

### Option C: Paid Coworking (€25-29)
1. Book **Base Coworking** or **WorkRepublic** day pass
2. Reserve a phone booth if possible
3. Guaranteed quiet, professional space
4. Worth it for an important interview
