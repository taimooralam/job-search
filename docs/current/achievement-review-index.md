## Interview Flow Reference

When we review an achievement, I'll ask these questions:

1. **Context & Role**: Your position, team size, main responsibilities
2. **The Challenge**: What problem led to this achievement?
3. **Your Actions**: What YOU specifically did (not the team)
4. **Measurable Outcomes**: Verify each metric - is it real? How measured?
5. **Metric Verification**: Before/after, data source
6. **Missing Context**: Any leadership/process/architecture aspects not captured?
7. **Keywords & Skills**: Technologies or skills to highlight?
